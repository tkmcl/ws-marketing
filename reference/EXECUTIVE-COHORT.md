## How to Use This Cohort

When testing messaging, content, or positioning:
1. Select 5-10 personas (or all)
2. Present the same stimulus to each (headline, value prop, email, etc.)
3. Each persona responds from their POV: interest level, objections, what resonates
4. Aggregate patterns and outliers
